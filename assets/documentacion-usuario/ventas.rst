Módulo de Ventas
================

Este módulo gestiona las ventas de servicios en el sistema. A continuación se detallan las opciones disponibles en el menú de ventas y cómo utilizarlas.

1. **Vender un Servicio**
   -----------------------
   Selecciona la opción "1" para registrar una nueva venta. El sistema te pedirá que selecciones el tipo de carga (Pasajero o Encomienda). Luego, se te solicitarán los datos necesarios para la venta, como el número de identificación del cliente, el código del servicio y la cantidad a vender. Asegúrate de ingresar datos válidos y confirmar la disponibilidad de los servicios antes de completar la venta. Después, puedes optar por imprimir una factura si lo deseas.

2. **Consultar Todas las Ventas**
   -------------------------------
   Elige la opción "2" para consultar y visualizar todas las ventas registradas en la base de datos. El sistema mostrará una lista completa de las ventas.

3. **Consultar un Dato de una Venta**
   ------------------------------------
   Selecciona la opción "3" para consultar información específica sobre una venta. El sistema te permitirá elegir qué dato deseas consultar (número de identificación del cliente, código del servicio, cantidad vendida) y te pedirá el número de la factura. La información correspondiente se mostrará en pantalla.

4. **Consultar Cantidad Total de Ventas**
   ----------------------------------------
   Utiliza la opción "4" para conocer la cantidad total de ventas registradas en la base de datos. El sistema te mostrará el número total de registros en la tabla de ventas.

5. **Consultar Registros por Número de Identificación del Cliente**
   -----------------------------------------------------------------
   Con la opción "5", puedes buscar ventas relacionadas con un número de identificación de cliente específico. Introduce el número de identificación del cliente y el sistema mostrará los registros correspondientes.

6. **Consultar Registros por Código del Servicio**
   --------------------------------------------------
   Usa la opción "6" para buscar ventas relacionadas con un código de servicio específico. Proporciona el código del servicio y el sistema mostrará los registros correspondientes.

7. **Borrar una Venta**
   ---------------------
   Selecciona la opción "7" para eliminar una venta específica. Introduce el número de la factura y confirma la acción. La venta será borrada de la base de datos.

8. **Borrar Toda la Tabla de Ventas**
   ------------------------------------
   Con la opción "8", puedes eliminar todos los registros de la tabla de ventas. Se te pedirá confirmar la acción antes de que la tabla sea completamente borrada.

9. **Imprimir Factura**
   ---------------------
   Utiliza la opción "9" para imprimir una factura. Introduce el número de la factura a imprimir, y el sistema recuperará los datos de la venta, cliente y servicio asociados a esa factura para generar el documento.

10. **Volver**
    -----------
    Selecciona la opción "10" para salir del menú de ventas y regresar al menú principal.

Para navegar por el menú, selecciona la opción deseada ingresando el número correspondiente y sigue las instrucciones que aparecen en pantalla.

**Nota:** Asegúrate de ingresar datos válidos y confirmar acciones críticas para evitar la pérdida accidental de información o errores en las ventas.
