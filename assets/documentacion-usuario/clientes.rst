Módulo de Gestión de Clientes
=============================

Este módulo permite gestionar los clientes registrados en el sistema. A continuación se detallan las opciones disponibles en el menú de gestión de clientes y cómo utilizarlas.

1. **Insertar un Nuevo Cliente**
   ------------------------------
   Para añadir un nuevo cliente a la base de datos, selecciona la opción "1" en el menú de clientes. El sistema te pedirá que ingreses la información del cliente, como nombre, apellido, dirección, teléfono y correo electrónico. Una vez que hayas proporcionado los datos, el cliente será registrado en el sistema.

2. **Consultar Todos los Clientes**
   ---------------------------------
   Elige la opción "2" para consultar y visualizar todos los clientes registrados en la base de datos. El sistema mostrará una lista completa de los clientes.

3. **Consultar un Dato de un Cliente**
   ------------------------------------
   Selecciona la opción "3" para consultar información específica de un cliente. El sistema te permitirá elegir qué dato deseas consultar (nombre, apellido, dirección, teléfono, correo electrónico) y te pedirá la identificación del cliente. La información correspondiente se mostrará en pantalla.

4. **Consultar Cantidad Total de Clientes**
   ------------------------------------------
   Utiliza la opción "4" para conocer la cantidad total de clientes registrados en la base de datos. El sistema te mostrará el número total de registros en la tabla de clientes.

5. **Consultar Registro por Nombre**
   ----------------------------------
   Con la opción "5", puedes buscar clientes por nombre. Introduce el nombre del cliente que deseas buscar y el sistema mostrará los registros correspondientes.

6. **Consultar Registro por Letra Inicial del Nombre**
   -----------------------------------------------------
   Usa la opción "6" para buscar clientes cuyo nombre comienza con una letra específica. Proporciona la letra inicial y el sistema mostrará los clientes que coincidan.

7. **Actualizar Nombre de un Cliente**
   ------------------------------------
   Para actualizar el nombre de un cliente existente, selecciona la opción "7". El sistema te pedirá el número de identificación del cliente y el nuevo nombre. El nombre del cliente será actualizado en la base de datos.

8. **Borrar un Cliente**
   ----------------------
   Elige la opción "8" para eliminar un cliente específico de la base de datos. Proporciona el número de identificación del cliente y confirma la acción. El cliente será borrado de la base de datos.

9. **Borrar Toda la Tabla de Clientes**
   --------------------------------------
   Con la opción "9", puedes eliminar todos los registros de la tabla de clientes. Se te pedirá confirmar la acción antes de que la tabla sea completamente borrada.

10. **Volver**
    -----------
    Selecciona la opción "10" para salir del menú de clientes y regresar al menú principal.

Para navegar por el menú, selecciona la opción deseada ingresando el número correspondiente y sigue las instrucciones que aparecen en pantalla.

**Nota:** Asegúrate de ingresar datos válidos y confirmar acciones críticas para evitar la pérdida accidental de información.
