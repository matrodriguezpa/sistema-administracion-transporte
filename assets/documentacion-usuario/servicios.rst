Módulo de Gestión de Servicios
==============================

Este módulo permite gestionar los servicios ofrecidos por el sistema. A continuación se describen las opciones disponibles en el menú de servicios y cómo usarlas.

1. **Insertar un Servicio**
   --------------------------
   Para insertar un nuevo servicio en la base de datos, selecciona la opción "1" en el menú de servicios. El sistema te pedirá que ingreses los detalles del servicio. Después de proporcionar la información requerida, el servicio se añadirá a la base de datos.

2. **Consultar Todos los Servicios**
   -----------------------------------
   Selecciona la opción "2" para consultar y visualizar todos los servicios disponibles en la base de datos. El sistema mostrará una lista completa de los servicios registrados.

3. **Consultar Fecha y Hora de Salida de Todos los Servicios**
   ------------------------------------------------------------
   Elige la opción "3" para consultar la fecha y hora de salida de todos los servicios. Esta opción te permitirá ver cuándo están programadas las salidas para todos los servicios.

4. **Consultar Puestos y Peso Máximos de Todos los Servicios**
   ------------------------------------------------------------
   Con la opción "4", podrás consultar la capacidad máxima en términos de puestos y peso para cada servicio. Esto te ayudará a conocer la capacidad de cada servicio ofrecido.

5. **Consultar un Dato de un Servicio**
   -------------------------------------
   Selecciona la opción "5" para consultar un dato específico de un servicio. El sistema te pedirá que elijas qué dato deseas consultar (nombre, origen, destino, precio, hora de salida, cantidad máxima de puestos, cantidad máxima de carga) y el código del servicio. La información correspondiente se mostrará en pantalla.

6. **Consultar Cantidad Total de Servicios**
   -------------------------------------------
   Utiliza la opción "6" para conocer la cantidad total de servicios registrados en la base de datos. El sistema mostrará el número total de registros en la tabla de servicios.

7. **Consultar Suma de los Precios de Venta de los Servicios**
   ------------------------------------------------------------
   Con la opción "7", puedes consultar la suma total de los precios de venta de todos los servicios. El sistema calculará y mostrará la sumatoria de los precios.

8. **Consultar Registro por Nombre**
   ----------------------------------
   Selecciona la opción "8" para buscar y consultar registros de servicios basados en el nombre. Ingresa el nombre del servicio que deseas buscar y el sistema te mostrará los registros correspondientes.

9. **Consultar Registro por Letra Inicial del Nombre**
   -----------------------------------------------------
   Usa la opción "9" para buscar servicios cuyos nombres comiencen con una letra específica. Proporciona la letra inicial y el sistema mostrará los servicios que coincidan.

10. **Actualizar el Nombre de un Registro**
    ----------------------------------------
    Para actualizar el nombre de un servicio existente, selecciona la opción "10". El sistema te pedirá el código del servicio y el nuevo nombre. El nombre del servicio será actualizado en la base de datos.

11. **Borrar un Registro de la Tabla de Servicios**
    --------------------------------------------------
    Elige la opción "11" para eliminar un registro específico de la tabla de servicios. Proporciona el código del servicio que deseas borrar y confirma la acción. El servicio será eliminado de la base de datos.

12. **Borrar Toda la Tabla de Servicios**
    --------------------------------------
    Con la opción "12", puedes borrar todos los registros en la tabla de servicios. Se te pedirá confirmar la acción antes de que la tabla sea completamente borrada.

13. **Volver**
    -----------
    Selecciona la opción "13" para salir del menú de servicios y regresar al menú principal.

Para navegar por el menú, selecciona la opción deseada ingresando el número correspondiente y sigue las instrucciones que aparecen en pantalla.

**Nota:** Asegúrate de ingresar datos válidos y confirmar acciones críticas para evitar la pérdida accidental de información.
