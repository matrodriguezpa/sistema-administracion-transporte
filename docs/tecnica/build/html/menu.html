Menú Principal¶

class Menu¶

Clase que representa un sistema de administración de transportes con menús para gestionar servicios, clientes y ventas.

Librerías:

  • sys: Utilizada para cerrar el programa desde la interfaz.

  • time: Utilizada para pausar el programa, proporcionando una breve demora.

Notas:

  • Cada método de menú contiene un bucle para manejar las opciones seleccionadas por el usuario.

  • Las operaciones de CRUD (Crear, Leer, Actualizar, Borrar) se gestionan a través de métodos específicos para servicios, clientes y ventas.

  • Se realizan validaciones para asegurar la integridad de los datos ingresados por el usuario.

Menu.menuPrincipal()¶

Muestra el menú principal del sistema.

Devuelve:

La opción seleccionada por el usuario.

Tipo del valor devuelto:

str

Menu.menuServicios(objetoConexion, objetoServicios)¶

Muestra el menú para gestionar la tabla de servicios.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • objetoServicios (Clase que maneja la tabla de servicios) – Objeto que maneja las operaciones en la tabla de servicios.

Devuelve:

None

Menu.menu(objetoConexion, objetoServicios, objetoVentas, objetoClientes, objetoMenu)¶

Inicia el menú principal y gestiona la creación de tablas y la navegación del usuario.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • objetoServicios (Clase que maneja la tabla de servicios) – Objeto que maneja las operaciones en la tabla de servicios.

  • objetoVentas (Clase que maneja la tabla de ventas) – Objeto que maneja las operaciones en la tabla de ventas.

  • objetoClientes (Clase que maneja la tabla de clientes) – Objeto que maneja las operaciones en la tabla de clientes.

  • objetoMenu (Clase Menu) – Instancia de la clase Menu para manejar la navegación entre menús.

Devuelve:

None
