Módulo gestion de clientes¶

class Clientes¶
Clase para gestionar la tabla clientes de la base de datos.

Atributos¶

  • noIdentificacionCliente: Número de identificación del cliente.

  • nombre: Nombre del cliente.

  • apellido: Apellido del cliente.

  • direccion: Dirección del cliente.

  • telefono: Número de teléfono del cliente.

  • correoElectronico: Correo electrónico del cliente.

Métodos¶

Clientes.crearTablaClientes(objetoConexion)¶

Crea la tabla “Clientes” en la base de datos si no existe.

Parámetros:

objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

None

Clientes.leerCliente()¶

Solicita al usuario los datos de un cliente y los retorna como una tupla.

Devuelve:

Datos del cliente ingresados por el usuario.

Tipo del valor devuelto:

tuple

Clientes.insertarTablaClientes(objetoConexion, miCliente)¶

Inserta un nuevo cliente en la tabla “Clientes”.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • miCliente (tuple) – Datos del cliente a insertar.

Devuelve:

None

Clientes.consultarTablaClientes1(objetoConexion)¶

Consulta y muestra todos los registros de la tabla “Clientes”.

Parámetros:

objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

None

Clientes.consultarTablaClientes2(objetoConexion, datoConsulta, noIdentificacionCliente)¶

Consulta un dato específico de un cliente.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • datoConsulta (str) – Nombre de la columna a consultar.

  • noIdentificacionCliente (int) – Número de identificación del cliente a consultar.

Devuelve:

Valor del dato consultado.

Tipo del valor devuelto:

str

Clientes.consultarTablaSClientes3(objetoConexion)¶

Consulta la cantidad total de registros en la tabla “Clientes”.

Parámetros:

objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

Cantidad total de registros.

Tipo del valor devuelto:

int

Clientes.consultarTablaClientes4(objetoConexion, dato, consulta)¶

Consulta registros en la tabla “Clientes” filtrados por un dato específico.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • dato (str) – Nombre de la columna por la cual filtrar.

  • consulta (str) – Valor del dato a buscar.

Devuelve:

Primer registro coincidente.

Tipo del valor devuelto:

tuple

Clientes.consultarTablaClientes5(objetoConexion, letraInicial)¶

Consulta registros en la tabla “Clientes” filtrados por la letra inicial del nombre.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • letraInicial (str) – Letra inicial para filtrar los nombres.

Devuelve:

Lista de registros que cumplen con el filtro.

Tipo del valor devuelto:

list

Clientes.actualizarTablaClientes(objetoConexion, nuevoNombre, noIdentificacionCliente)¶

Actualiza el nombre de un cliente en la tabla “Clientes”.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • nuevoNombre (str) – Nuevo nombre para el cliente.

  • noIdentificacionCliente (int) – Número de identificación del cliente a actualizar.

Devuelve:

None

Clientes.borrarRegistroTablaClientes(objetoConexion, noIdentificacionCliente)¶

Elimina un registro de la tabla “Clientes”.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • noIdentificacionCliente (int) – Número de identificación del cliente a eliminar.

Devuelve:

None

Clientes.borrarTablaClientes(objetoConexion)¶

Elimina la tabla “Clientes” de la base de datos.

Parámetros:

objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:
