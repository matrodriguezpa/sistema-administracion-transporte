Clase Ventas¶

La clase Ventas gestiona las operaciones relacionadas con la tabla de ventas en una base de datos y el envío de facturas por correo electrónico.

class Ventas¶

Maneja la creación, lectura, actualización y eliminación de registros de ventas, así como la capacidad para enviar facturas por correo electrónico.

Atributos¶

  • noFactura: Número único de la factura para identificar cada venta.

  • noIdentificacionCliente: Número de identificación del cliente que realiza la compra.

  • codigoServicio: Código único del servicio que se está vendiendo.

  • cantidadVendida: Cantidad de unidades del servicio vendidas.

Métodos¶

crearTablaVentas(self, con)¶

Crea la tabla Ventas en la base de datos si no existe.

Parámetros:

con (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

None

leerVenta(self, objetoConexion, objetoVentas)¶

Lee y genera un nuevo registro de venta solicitando al usuario los datos necesarios.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • objetoVentas (Ventas) – Instancia de la clase que maneja la consulta de ventas.

Devuelve:

tuple de los datos de la venta generada

añadirServicioAVender(self, objetoConexion, miVenta)¶

Añade un nuevo registro de venta a la base de datos.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • miVenta (tuple) – Datos de la venta a insertar.

Devuelve:

None

consultarTablaVentas1(self, objetoConexion)¶

Consulta y muestra todos los registros en la tabla Ventas.

Parámetros:

objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

None

consultarTablaVentas2(self, objetoConexion, dato, noFactura)¶

Consulta un dato específico en la tabla Ventas basado en el número de factura.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • dato (str) – El dato a consultar.

  • noFactura (str) – Número de factura a buscar.

Devuelve:

El valor del dato consultado o None si no se encuentra.

consultarTablaVentas3(self, con)¶

Consulta el número total de registros en la tabla Ventas.

Parámetros:

con (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

Número total de registros en la tabla.

consultarTablaVentas4(self, objetoConexion, dato, datoConsulta)¶

Consulta registros en la tabla Ventas basado en un dato específico.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • dato (str) – El dato a consultar.

  • datoConsulta (str) – Valor del dato a buscar.

Devuelve:

Lista de registros que coinciden con la consulta o None si no se encuentran.

consultarTablaVentas5(self, objetoConexion)¶

Consulta la suma total de las cantidades vendidas en la tabla Ventas.

Parámetros:

objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

La suma total de las cantidades vendidas o 0 si no hay ventas.

borrarRegistroTablaVentas(self, objetoConexion, noFactura)¶

Borra un registro específico en la tabla Ventas basado en el número de factura.

Parámetros:

  • objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

  • noFactura (str) – Número de factura del registro a borrar.

Devuelve:

None

borrarTablaVentas(self, objetoConexion)¶

Borra toda la tabla Ventas.

Parámetros:

objetoConexion (Objeto de conexión) – Objeto de conexión a la base de datos.

Devuelve:

None

imprimirFactura(self, miVenta, miCliente, miServicio)¶

Envía la factura de una venta por correo electrónico.

Parámetros:

  • miVenta (tuple) – Datos de la venta.

  • miCliente (tuple) – Datos del cliente.

  • miServicio (tuple) – Datos del servicio.

Devuelve:

None
