Librerías del Sistema
=====================

Base de Datos
-------------
**sqlite3**

La librería `sqlite3` es una interfaz para la base de datos SQLite, que es un motor de base de datos ligero y autónomo. Esta librería permite crear, conectar y manipular bases de datos SQLite desde aplicaciones Python. Es ideal para aplicaciones pequeñas y medianas que no requieren una base de datos más compleja o con mayores recursos.

Manejo de Fechas y Horas
-------------------------
**datetime**

La librería `datetime` proporciona clases para manipular fechas y horas en Python. Permite la creación de objetos de fecha y hora, y ofrece funcionalidades para comparar, calcular diferencias y formatear estas entidades. Es ampliamente utilizada para tareas que requieren gestión precisa del tiempo.

Cerrar el Programa
-------------------
**sys**

La librería `sys` ofrece acceso a algunas variables y funciones que interactúan con el intérprete de Python. Entre sus funcionalidades, permite cerrar el programa usando `sys.exit()`, lo cual es útil para terminar un script de manera controlada.

**time**

La librería `time` proporciona funciones para trabajar con el tiempo. Permite medir el tiempo transcurrido, pausar la ejecución del programa con `time.sleep()`, y manipular datos relacionados con el tiempo. Es útil para tareas que requieren temporización o control de la duración de ciertas operaciones.

Envío de Correos
----------------
**smtplib**

La librería `smtplib` define una interfaz para enviar correos electrónicos utilizando el protocolo SMTP (Simple Mail Transfer Protocol). Permite la conexión con un servidor SMTP y el envío de correos electrónicos desde una cuenta configurada.

**getpass**

La librería `getpass` se utiliza para leer contraseñas u otros datos sensibles sin mostrar el texto en la consola. Es útil para solicitar credenciales de manera segura en una aplicación.
